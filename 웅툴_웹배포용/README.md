# 웅툴 — 업무용 PDF 도구

반복되는 PDF 작업을 브라우저에서 처리하는 정적 웹앱입니다.

## 특징

- 압축을 직접 풀 필요가 없습니다.
- `label_1`, `label_2`, `label_10`처럼 숫자를 고려해 정렬합니다.
- UPS A4·Letter 라벨을 10×15cm 라벨 용지로 변환합니다.
- UPS Backup Document 페이지를 자동 감지해 제외합니다.
- 인보이스 XLS의 SKU를 Google DB와 매칭해 Location을 입력합니다.
- Remark와 Description을 생략 없이 표시하는 작업자용 A4 디자인으로 재구성합니다.
- 동일한 피킹리스트를 Excel과 PDF 두 형식으로 내려받을 수 있습니다.
- 모든 처리는 사용자의 브라우저 안에서 이루어집니다.
- 파일을 서버에 업로드하거나 보관하지 않습니다.
- 별도의 백엔드와 데이터베이스가 필요 없습니다.

## 배포 방법

### Netlify Drop

1. 이 폴더의 파일을 모두 선택해 ZIP으로 압축합니다.
2. [Netlify Drop](https://app.netlify.com/drop)에 ZIP을 끌어놓습니다.
3. 생성된 주소를 팀원에게 공유합니다.

### Vercel

Vercel에서 새 프로젝트를 만든 뒤 이 폴더를 배포합니다. 프레임워크 설정은 `Other`, 빌드 명령은 비워 두고 출력 폴더는 `.`으로 설정합니다.

### 사내 웹서버

이 폴더 전체를 웹서버의 공개 폴더에 복사하면 됩니다. `index.html`이 시작 페이지입니다.

## 로컬 확인

보안 정책상 `index.html`을 직접 더블클릭하기보다 간단한 로컬 웹서버를 통해 여는 것을 권장합니다.

```bash
python -m http.server 8080
```

그다음 브라우저에서 `http://localhost:8080`을 엽니다.
